# HuMobi 快速开始（Quick Start）

本指南带你用最少的代码完成 HuMobi 的典型工作流：读取轨迹数据 → 预处理 → 聚合 → 计算指标 → 预测。关键术语均标注英文。

---

## 1. 安装（Installation）

```bash
pip install HuMobi
```

建议创建虚拟环境。主要依赖包括 pandas、geopandas、scipy、numpy、scikit-learn、shapely、numba、biopython、geofeather 以及 tensorflow-gpu（仅深度学习预测需要）。

---

## 2. 读取数据（Data Reading）

HuMobi 的核心数据结构是 **轨迹框架（TrajectoriesFrame）**，它继承自 GeoPandas 的 `GeoDataFrame`，并使用 `MultiIndex`（`user_id` / `datetime`）组织个体移动轨迹。

### 2.1 从文件读取

```python
from humobi.structures import trajectory as tr

in_path = "brightkite_sample.tsv"
df = tr.TrajectoriesFrame(
    in_path,
    {
        'names': ['id', 'datetime', 'lat', 'lon', 'place'],
        'delimiter': '\t',
        'crs': 4326          # 坐标参考系（CRS, Coordinate Reference System）
    }
)
```

### 2.2 从 DataFrame 转换

```python
import pandas as pd

df = pd.read_csv(in_path, names=['id', 'datetime', 'lat', 'lon', 'place'], delimiter='\t')
df = tr.TrajectoriesFrame(df)
```

### 2.3 保存与读取

```python
# 保存为 CSV
df.to_csv("converted_sample.csv")

# 保存为 GeoFeather
df.to_geofeather("sample.feather")

# 从 GeoFeather 读取（类方法）
df = tr.TrajectoriesFrame.from_geofeather("sample.feather")

# 保存为 Shapefile（注意：时间戳会被转为字符串）
df.to_shapefile("sample.shp")
```

### 2.4 坐标转换（CRS Reprojection）

```python
# 从 WGS84 (EPSG:4326) 转换到 Web Mercator (EPSG:3857)，便于按米计算距离
df_metric = df.to_crs(dest_crs=3857, cur_crs=4326)
```

## 3. 数据选择（Data Selection）

### 3.1 选择单个或多个用户

```python
# 选择用户 ID 为 0 的轨迹
one_user = df.uloc(0)

# 获取所有用户 ID
users_list = df.get_users()

# 选择多个用户
many_users = df.uloc(users_list)
```

### 3.2 基于用户统计的过滤

```python
from humobi.tools.user_statistics import *

# 缺失记录比例（Fraction of empty records）
frac = fraction_of_empty_records(df, resolution='1H')

# 每个用户的总记录数
count = count_records(df)

# 轨迹持续时间（Trajectory duration）
duration = user_trajectories_duration(df, resolution='1D', count_empty=False)

# 最长连续记录数
consecutive = consecutive_record(df, resolution='1H')
```

### 3.3 综合过滤示例

仅保留：缺失率 < 60%、轨迹时长 > 6 天、总记录数 ≥ 100 的用户。

```python
level1 = set(frac[frac < 0.6].index)
level2 = set(duration[duration > 6].index)
level3 = set(count[count >= 100].index)

selection = level1.intersection(level2).intersection(level3)
df_sel = df.uloc(list(selection))

# 按时间戳排序并重新构造 TrajectoriesFrame
df_sel = df_sel.sort_index(level=1)
df_sel = tr.TrajectoriesFrame(df_sel, {'crs': df.crs, 'geom_cols': df.geom_cols})
```

---

## 4. 停留点检测（Stop Detection）

使用 `stop_detection` 函数检测停留点（stay-points）。该函数内部会临时转换为 `EPSG:3857` 以按米计算距离，并使用多线程加速。

```python
from humobi.preprocessing.filters import stop_detection
from humobi.tools.processing import start_end

# distance_condition: 空间阈值，单位米
# time_condition: 时间阈值
stops = stop_detection(df_sel, distance_condition=300, time_condition='10min')

# 仅保留停留点
df_stops = stops[stops['is_stop'] == True].drop_duplicates()

# 压缩停留记录：为每个停留位置添加访问开始/结束时间
df_stops = start_end(df_stops)

# 保存中间结果
df_stops.to_csv("preprocessed_stops_sample.csv")
```

---

## 5. 数据聚合（Data Aggregation）

预处理后的停留点需要经过**空间聚合（spatial aggregation）**得到停留区（stay-regions / labels），再经过**时间聚合（temporal aggregation）**得到规则的时间箱（time-bins）序列。

### 5.1 转换为公制 CRS

```python
df_stops = df_stops.to_crs(dest_crs="EPSG:3857")
```

### 5.2 空间聚合：DBSCAN 聚类

```python
from humobi.preprocessing.spatial_aggregation import ClusteringAggregator
from sklearn.cluster import DBSCAN

clust_agg = ClusteringAggregator(
    DBSCAN,
    **{"eps": 300, "min_samples": 2}  # eps 单位为米
)
df_clust = clust_agg.aggregate(df_stops)
```

### 5.3 空间聚合：规则网格

```python
from humobi.preprocessing.spatial_aggregation import GridAggregation

grid_resolution = 1000  # 网格边长，单位米
grid_agg = GridAggregation(grid_resolution)
df_grid = grid_agg.aggregate(df_stops, parallel=False)
```

### 5.4 时间聚合：下一时间箱（Next Time-bin）

```python
from humobi.preprocessing.temporal_aggregation import TemporalAggregator

time_unit = '1H'  # 1 小时为一个时间箱
time_agg = TemporalAggregator(time_unit)

# parallel=True 使用多线程，可加速
df_time = time_agg.aggregate(df_clust, parallel=True)

# fill_method 可填 'ffill' / 'bfill'，drop_empty=True 删除空时间箱
df_time_filled = time_agg.aggregate(df_clust, fill_method='ffill', drop_empty=False)
```

### 5.5 下一地点序列（Next Location Sequence）

删除同一停留区内的连续重复记录：

```python
from humobi.preprocessing.filters import next_location_sequence

df_seq = next_location_sequence(df_time)
```

## 6. 计算指标（Metrics）

### 6.1 个体指标（Individual Metrics）

```python
from humobi.measures.individual import *
from humobi.measures.collective import *

# 不同地点数量
distinct = num_of_distinct_locations(df_seq)

# 访问频率
vfreq = visitation_frequency(df_seq)

# 跳跃长度（Jump lengths）
jumps = jump_lengths(df_seq)

# 非零出行次数
nonzero = nonzero_trips(df_seq)

# 自转移次数（Self-transitions）
st = self_transitions(df_seq)

# 等待时间
wt = waiting_times(df_seq, time_unit='h')

# 质心（Center of mass）
com = center_of_mass(df_seq)

# 回转半径（Radius of gyration）
rog = radius_of_gyration(df_seq, time_evolution=False)

# 均方位移（Mean square displacement）
msd = mean_square_displacement(df_seq, time_evolution=False)

# 返回时间（Return time）
rt = return_time(df_seq, time_unit='H')
rt_by_place = return_time(df_seq, by_place=True)
```

### 6.2 熵与可预测性（Entropy & Predictability）

```python
# 随机熵（Random entropy）与其可预测性
re = random_entropy(df_seq)
rp = random_predictability(df_seq)

# 非相关熵（Uncorrelated entropy）与其可预测性
ue = unc_entropy(df_seq)
up = unc_predictability(df_seq)

# 真实熵（Real entropy）与其可预测性（需要 GPU，缺失率 > 90% 时不可用）
# re_e = real_entropy(df_seq)
# rp_e = real_predictability(df_seq)
```

### 6.3 集体指标（Collective Metrics）

```python
# 出行距离分布
dist = dist_travelling_distance(df_seq)

# 聚合单元之间的流量（flows）
flow = flows(df_seq, flows_type='all')  # 'all' | 'incoming' | 'outgoing'
```

## 8. 下一地点预测（Next-Location Prediction）

### 8.1 数据集划分

```python
from humobi.predictors.wrapper import *
from humobi.predictors.deep import *

# split_ratio: 测试集比例
# horizon: 历史窗口长度
# n_splits: 交叉验证折数
data_splitter = Splitter(markovian_seq, split_ratio=0.2, horizon=5, n_splits=2)

test_X = data_splitter.test_frame_X
test_Y = data_splitter.test_frame_Y
cv_data = data_splitter.cv_data
```

### 8.2 基线方法：TopLoc

```python
toploc_results = TopLoc(
    train_data=cv_data,
    test_data=[test_X, test_Y]
).predict()

# 返回 (predictions, accuracy)
predictions, accuracy = toploc_results
```

### 8.3 马尔可夫链（Markov Chain）

```python
# state_size: 马尔可夫链阶数
# update: 是否根据预测结果更新模型
# online: 预测时是否能看到测试集最后 state_size 个符号
MC1 = markov_wrapper(markovian_seq, test_size=0.2, state_size=1, update=False, online=True)
MC2 = markov_wrapper(markovian_seq, test_size=0.2, state_size=2, update=False, online=True)

predictions, accuracy = MC2
```

### 8.4 scikit-learn 分类器

```python
from sklearn.ensemble import RandomForestClassifier

clf = RandomForestClassifier
predic = SKLearnPred(
    algorithm=clf,
    training_data=cv_data,
    test_data=[test_X, test_Y],
    param_dist={
        'n_estimators': list(range(500, 5000, 500)),
        'max_depth': [None, 2, 4, 6, 8]
    },
    search_size=5,
    cv_size=5,
    parallel=False
)

predic.learn()
predic.test()

scores = predic.scores
predictions = predic.predictions
```

### 8.5 深度学习：GRU


## 9. 可重复性指标（Repeatability Metrics）

这些指标基于训练集（train frame）和测试集（test frame），用于解释预测精度差异。

```python
from humobi.measures.individual import *
import pandas as pd

X = pd.concat([cv_data[0][0], cv_data[0][2]])  # 训练 + 验证特征

DR = repeatability_dense(train_frame=X, test_frame=Y)
```

## 10. 完整流程示例

```python
from humobi.structures import trajectory as tr
from humobi.tools.user_statistics import *
from humobi.preprocessing.filters import stop_detection, next_location_sequence
from humobi.preprocessing.spatial_aggregation import ClusteringAggregator
from humobi.preprocessing.temporal_aggregation import TemporalAggregator
from humobi.measures.individual import *
from sklearn.cluster import DBSCAN

# 1. 读取
df = tr.TrajectoriesFrame(
    "brightkite_sample.tsv",
    {'names': ['id', 'datetime', 'lat', 'lon', 'place'], 'delimiter': '\t', 'crs': 4326}
)

# 2. 过滤用户
frac = fraction_of_empty_records(df, '1H')
count = count_records(df)
duration = user_trajectories_duration(df, '1D', count_empty=False)

selected = set(frac[frac < 0.6].index) \
    .intersection(set(count[count >= 100].index)) \
    .intersection(set(duration[duration > 6].index))

df = df.uloc(list(selected)).sort_index(level=1)
df = tr.TrajectoriesFrame(df, {'crs': 4326})

# 3. 检测停留点
df = stop_detection(df, distance_condition=300, time_condition='10min')
df = df[df['is_stop'] == True].drop_duplicates()

# 4. 聚合
df = df.to_crs(dest_crs="EPSG:3857")
df = ClusteringAggregator(DBSCAN, **{"eps": 300, "min_samples": 2}).aggregate(df)
df = TemporalAggregator('1H').aggregate(df, parallel=True)
df = next_location_sequence(df)

# 5. 计算指标
print(num_of_distinct_locations(df).head())
print(random_predictability(df).head())
```

## 11. 注意事项（Notes）

- `TrajectoriesFrame` 基于 pandas `DataFrame`，因此大多数 pandas 选择、过滤、分组操作都适用。
- 时间聚合的时间箱从午夜开始。
- 真实熵（real entropy / predictability）需要 GPU 环境，且当缺失率 > 90% 时不可计算。
- 移动模型（mobility model）模块仍在开发中，当前不建议用于生产。
